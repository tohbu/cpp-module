## staticメンバー関数について

💬 一言でまとめると：

通常メンバ関数は「インスタンスが所有する処理」

staticメンバ関数は「クラス全体に属する処理（全員で共有）」

```cpp
class Sample {
public:
    void hello();
    int value;
};

        ┌────────────┐
        │   Class: Sample   │
        │──────────────────│
        │ hello() ← 定義だけ（コードは共通） │
        └────────────┘
               │
───────────────┼──────────────────────────
               │
       (インスタンスごとに独立した変数)
               ▼

┌────────────┐      ┌────────────┐
│ Sample A    │      │ Sample B    │
│────────────│      │────────────│
│ value = 10 │      │ value = 99 │
│ hello() → Aを操作 │ │ hello() → Bを操作 │
└────────────┘      └────────────┘
```

```cpp
class Sample {
public:
    static void helloStatic();
    static int count;
};

        ┌────────────┐
        │   Class: Sample   │
        │──────────────────│
        │ static int count  │ ← クラス全体で1つだけ存在
        │ helloStatic()     │ ← クラス全体で1つだけ存在
        └────────────┘
               │
───────────────┼──────────────────────────
               │
       (インスタンスはそれを共有する)
               ▼

┌────────────┐      ┌────────────┐
│ Sample A    │      │ Sample B    │
│────────────│      │────────────│
│ (countなし) │      │ (countなし) │
│ helloStatic() ⇢ Sample::helloStatic │
└────────────┘      └────────────┘
```
