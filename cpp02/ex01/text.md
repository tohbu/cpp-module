| 変換の種類       | 例                                  | 結果   |
| ----------- | ---------------------------------- | ---- |
| 数値型どうしの変換   | `float f = static_cast<float>(i);` | ✅ OK |
| 型昇格・縮小      | `double → int`, `char → int`       | ✅ OK |
| 継承でのアップキャスト | `Derived* → Base*`                 | ✅ OK |
| ユーザー定義変換    | `operator int()` がある型              | ✅ OK |

| 危険な操作           | 例                   | 結果                  |
| --------------- | ------------------- | ------------------- |
| 無関係なポインタ変換      | `int* → double*`    | ❌ コンパイルエラー          |
| const除去         | `const int* → int*` | ❌ コンパイルエラー          |
| ダウンキャスト（安全確認なし） | `Base* → Derived*`  | ❌ dynamic_castを使うべき |
